CheckDropDown - на каждый Dropdown. В поле Right Variant указать правильный вариант для проверки

CheckInputField - на каждый InputField. В поле Right Variant указать правильный вариант для проверки

TechCardChecker - на любой объект активный (в момент проверки) объект сцены (желательно создать новый пустой объект, который точно будет активен)
InputFields - массив структур, который нужно заполнить. InputField - InputField соответственно. FieldName - имя, которое нужно добавить в сообщение,
если поле неправильное
Dropdowns - массив структур, который нужно заполнить. Dropdown - Dropdown соответственно. FieldName - имя, которое нужно добавить в сообщение,
если поле неправильное
CreateMessage() - функция, которая возвращает message (список неправильных полей в формате:"FieldName1\nFieldName2\n")

CheckProcesses - на любой объект активный (в момент проверки) объект сцены (желательно создать новый пустой объект, который точно будет активен)
dropDowns - массив DropDown (закинуть каждый dropdown)
rightAnswers - массив строк (в каждую строку закидывать один правильный ответ) (ВАЖНО!!! Символ в символ, всё должно быть точно)
